# Om Bhadke - Portfolio Website

A modern, responsive portfolio website showcasing graphic design and game development work, built with HTML, CSS, and JavaScript.

## About

Hi, I'm Om Bhadke! I'm a passionate graphic designer and game developer who creates visually stunning and engaging digital experiences. This portfolio showcases my work in UI/UX design, game development, and creative projects.

## Features

- **Responsive Design**: Works on all devices (desktop, tablet, mobile)
- **Smooth Scrolling**: Navigation links scroll smoothly to sections
- **Interactive Elements**: Hover effects, animations, and form validation
- **Modern UI**: Clean design with gradient backgrounds and card layouts
- **Contact Form**: Basic form with validation (ready for backend integration)

## Sections

1. **Hero**: Introduction with call-to-action button
2. **About**: Personal information and creative background
3. **Skills**: Design and development skills organized by category
4. **Projects**: Showcase of graphic design and game development work
5. **Contact**: Contact information and contact form

## Skills Highlighted

- **Graphic Design**: Adobe Photoshop, Illustrator, Figma, UI/UX Design, Brand Identity
- **Game Development**: Unity, Unreal Engine, C#, Game Design, 3D Modeling
- **Tools**: Blender, After Effects, Git, VS Code, HTML/CSS

## Projects Featured

- Indie Game Adventure (Unity 2D Platformer)
- Brand Identity Design Package
- 3D Character Modeling Collection

## Running the Website

1. Open `index.html` in a web browser
2. Or run a local server:
   ```bash
   python -m http.server 8000
   ```
   Then visit `http://localhost:8000`
